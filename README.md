# UE5_Plugins_GamepadLight
Simple UI Navigation for Gamepad and Keyboard【适配Xbox手柄和键盘的简易UI导航】
=

_无需过多更改UI和其他设置，简易的配置，自定义的键位触发_  
_让手柄和键盘都可以快速使用上控件UI的流畅导航_

还没有做PS手柄的适配（因为没买）虚幻原生只支持Xbox。
教程在bilibili搜
